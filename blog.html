Important FAQ

What is the Document Object Model(DOM) ?

Document Object Model is a programming interface for web documents. It treats a HTML document as a tree structure so that the nodes can be dynamically accessed and updated

How Do you select an element from the DOM ?

getElementById() and querySeloector() is used to select an element in DOM.

How do you manipulate an element's attribute and styles using DOM ?

There are several ways to manipulate an element's styles and attributes such as: change the content of an element, create,add and remove of an elements, insert element at aspecific position etc. For example remove() method can remove a specific class, add() method can add a specific class in a classList.

Why is event delegation important in DOM and how do we use it ?

Event delegation is the process of using event propagation to handle events at a higher level in the DOM than the element on which the event originated.It is useful because It allows us to attach a single event listener for elements that exist now or in the future.
